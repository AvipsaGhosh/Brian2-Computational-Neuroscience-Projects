# Working Memory Impairment via NMDA Hypofunction
### A Computational Neuroscience Model in Brian2

---

## Overview
This project implements a biologically inspired spiking neural network that demonstrates how alterations in recurrent cortical connectivity can impair working memory. The model is implemented using the Brian2 simulator and is based on the attractor network principles proposed by Wong and Wang (2006), with additional simplifications suitable for educational purposes.

The network illustrates how persistent neural activity, believed to underlie working memory in the prefrontal cortex, emerges from recurrent excitation and how weakening NMDA-mediated synaptic transmission destabilizes these attractor states.

Rather than attempting to simulate schizophrenia as a whole, this project focuses on one computational hypothesis associated with cognitive impairment: **NMDA receptor hypofunction leading to reduced stability of cortical working memory circuits.**

---

## Biological Motivation
Working memory refers to the brain's ability to temporarily maintain information after an external stimulus has disappeared.
* **Persistent Activity:** Experimental recordings from neurons in the prefrontal cortex show that many neurons continue firing during the delay period of working memory tasks, even after the stimulus has been removed. This sustained firing is believed to result from recurrent excitatory connections within cortical microcircuits.
* **Synaptic Mechanism:** One influential hypothesis proposes that slow NMDA receptor-mediated excitation maintains these persistent activity states.
* **Hypofunction Hypothesis:** Numerous computational studies have suggested that reduced NMDA receptor function weakens recurrent excitation, making these attractor states unstable and impairing working memory performance.

This project recreates this phenomenon using a simplified spiking neural network.

---

## Model Architecture
The network consists of two interacting neuronal populations:
1. **Excitatory neurons**
2. **Inhibitory neurons**

Each neuron is modeled as a stochastic leaky integrate-and-fire neuron. The simulation includes:
* Membrane potential dynamics
* Slow NMDA-mediated excitatory synapses
* Fast GABA-mediated inhibitory synapses
* Gaussian membrane noise
* Background cortical activity generated by a Poisson process

Together these components create a biologically plausible recurrent cortical circuit capable of exhibiting persistent activity.

---

## Simulation Protocol
Each simulation run lasts for 1500 ms and is divided into three distinct phases:

| Phase | Duration | Description |
| :--- | :--- | :--- |
| **Phase 1 — Rest** | 200 ms | No external stimulus is applied. The network settles into its spontaneous baseline activity. |
| **Phase 2 — Cue Presentation** | 300 ms | An external current is injected into the excitatory population. This stimulus activates the recurrent excitatory network and initiates persistent activity. |
| **Phase 3 — Delay Period** | 1000 ms | The external input is removed. If recurrent excitation is sufficiently strong, the network maintains activity internally through positive feedback. If weakened, activity decays. |

---

## Healthy vs NMDA Hypofunction Conditions
The simulation includes a Boolean configuration switch: `is_healthy = True`. Changing this value modifies several biologically relevant parameters.

### Healthy Network
* Strong recurrent NMDA excitation
* Balanced inhibitory feedback
* Lower intrinsic membrane noise
* **Expected behavior:** Persistent delay-period firing, stable attractor dynamics, and robust working memory maintenance.

### NMDA Hypofunction Network
* Reduced recurrent NMDA conductance
* Weaker inhibitory recruitment
* Increased membrane noise
* **Expected behavior:** Persistent activity becomes unstable, delay-period firing gradually decays, and there is a reduced robustness of the working memory attractor.

---

## Dynamical Interpretation
The network behaves as an attractor system:
* During stimulus presentation, recurrent excitation drives the network toward a stable high-activity state.
* In the **healthy condition**, this attractor is sufficiently deep that neural activity remains elevated after stimulus removal.
* Under **NMDA hypofunction**, recurrent excitation is weakened, making the attractor shallower. As a result, stochastic fluctuations and inhibitory feedback gradually push the system back toward its resting state.

This phenomenon is often described as a *leaky* or *metastable* attractor, representing reduced stability of persistent cortical activity.

---

## Simulation Outputs
The project produces two primary visualizations:
1. **Spike Raster:** Displays the firing activity of individual excitatory neurons over time. Healthy networks exhibit sustained coordinated firing throughout the delay period, while networks with NMDA hypofunction show reduced and gradually decaying delay-period activity.
2. **Population Firing Rate:** Displays the average firing rate of the excitatory population. Healthy networks maintain an elevated firing rate after stimulus removal, whereas networks with NMDA hypofunction exhibit significantly lower persistent firing.
